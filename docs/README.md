This directory mixes two different documentation layers, and it helps to keep them separate mentally.

## 2. Drafting and provenance trail

The top-level `*_draft.md` files and planning notes are intentionally retained as writing provenance. They are useful for collaborators, but they are not the canonical manuscript.

If you want the paper itself, go straight to:

- `../paper/main.tex`
- `../paper/main.pdf`

If you want the current paper-facing empirical bundle, pair the paper with:

- `../outputs/analysis/final_combined_v2/analysis_report.md`
- `../outputs/analysis/final_combined_v2/main_text_direct_contrasts.csv`
