# Docs Map

This directory contains two kinds of documents:

- `final_revision/`
  Submission-facing notes that match the current paper and released artifacts.
- top-level `*_draft.md` and planning notes
  Intermediate drafting history kept for provenance and writing traceability.

If you want the current release-facing documents first, start with:

- `final_revision/appendix_draft.md`
- `final_revision/main_text_stats_table.md`
- `final_revision/appendix_stats_table.md`
- `final_revision/reviewer_risk_memo_final.md`
- `paper/main.tex`
